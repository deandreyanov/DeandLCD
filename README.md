RST	 D7 - перезагрузка
CE	 D6 - разрешение на ввод данных
DC	 D5 - выбор режима ввода данных

CLK	 D3 - тактирование SPI интерфейса
VCC	 3.3В - питание. Диапазон напряжений от 2.7 до 3.3 вольт
GND	 GND
